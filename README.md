# README.md
## Running Server
(_Service used Gin for HTTP requests as well as CORS config._)
1. Install dependencies
	`go get .`

2. Start the application
	`go run .`

## Running Client
1. Install dependencies
	`npm install`

2. Start the application
	`npm start`
